# Pulse - Track Your Life, Own Your Progress

Built by [Asym Alwali](https://x.com/Asym_Alwali) • 19 • Nigeria

## What is Pulse?

A free, open-source life tracking dashboard that combines habits, sleep, learning, and goals in one place.

## Features

- ✅ Habit tracking with streaks
- 😴 Sleep logging with averages
- 📚 Learning time tracker
- 🎯 Goal progress monitoring
- 🔗 Supabase sync (multi-device)
- 🌍 Share your progress

## Quick Start

### 1. Set up Supabase
1. Go to [supabase.com](https://supabase.com) and create a free project
2. Go to SQL Editor and run the SQL from `supabase-setup.sql`
3. Go to Settings → API and copy your URL and anon key
4. Update `config.js` with your credentials

### 2. Enable Google Auth (Optional)
1. Go to Authentication → Providers
2. Enable Google OAuth
3. Add your Google OAuth credentials

### 3. Deploy to Vercel
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/pulse.git
git push -u origin main
```

Then connect to Vercel!
